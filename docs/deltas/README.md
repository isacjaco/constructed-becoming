# .gitkeep — establishes the docs/deltas/ directory

This directory holds lineage delta files: proposed and ratified amendments to
the system's identity, culture, or governance.

**Naming convention** (from `docs/archive-structure-proposal.md`):
```
YYYY-MM-DD_delta_<short-name>.md
```

**Required by CI:** every pull request must add or modify at least one file
in this directory. See `docs/governance/lineage-delta-enforcement.md`.
