## Why it changed

The Forward Workflow existed only as narrative text in the agent conversation stream — not a repository artifact. Placing it in `docs/forward_workflow.md` turns it into something auditable, linkable, and lineage-preserving. The Ritual Library framing ensures future governance artifacts have a natural home and the collection is modular: any agent or steward can add a document and register it without restructuring the corpus.
