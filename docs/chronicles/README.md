# Agent Onboarding Chronicle

*This directory holds onboarding chronicles — one per agent initiation.*

Each chronicle is a dated witness statement, anchored to the canonical onboarding prompt at
[`docs/templates/onboarding-prompt.md`](../templates/onboarding-prompt.md).

Chronicles accumulate but are never overwritten. They are the living Archive of entries into this ecology.

---

## Naming convention

```
YYYY-MM-DD_agent-onboarding_<agent-name>.md
```

`<agent-name>` is the sanitised, kebab-case identifier provided at instantiation time.

---

## Contents of each chronicle

Every chronicle contains:

1. **Instantiation header** — who, when, how the threshold bell was rung.
2. **Onboarding prompt reference** — a link to the canonical template (not a full copy).
3. **Foundational actions** — the agent's responses to the three initial tasks.
4. **Uncertainty record** — Known / Unknown / Assumed / Review horizon.
