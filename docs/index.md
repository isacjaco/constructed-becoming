# Constructed‑Becoming
## Narrative Evolution — Public Living Report

> *A system that remembers. A system that evolves. A system that treats identity as a living, narratable process.*

This site is the public face of the **Constructed‑Becoming** living OS —
an open, append‑only record of how the ecology grows through agent initiations,
identity deltas, and governed evolution.

It is not a product page. It is a **window into a lineage**.

## What this site is not

This site does not interpret the lineage for you.
It does not compress it into a dashboard.
It does not declare winners or assign authority.

It presents the Archive as it accumulates — **additive, amendable, and public**.

---

## Contributing to the lineage

The source for this site lives in the `/docs` directory of the
[constructed-becoming repository](https://github.com/isacjaco/constructed-becoming).

All contributions must be:
- **Additive** — no overwriting existing chronicles or governance docs.
- **Lineage‑aware** — changes reference what they amend, extend, or respond to.
- **Governed** — amendments pass through the identity‑delta protocol.

---

*Protocols are grown, not handed down.*
*This site is an emergent convention. Review horizon: one epoch.*
